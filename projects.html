Fruit Collector Game

Description: A browser-playable arcade-style game built with Python and Pygame, compiled using Pygbag for the web. Dodge bombs and collect fruits to win!

Future Plans:

  • Add background music and sound effects
  • Create animated fruit sprites
  • Introduce power-ups like slow-mo and double score
  • Track high scores with a leaderboard
  • Make it mobile-friendly

Asteroid Blaster

Description: A browser-playable asteroid shooting game built with Python and Pygame, compiled for the web using Pygbag. Dodge asteroids and blast them to survive!

Features:

  • Player momentum and rotation physics
  • Projectile shooting and asteroid splitting
  • background music and sound effects
  • Dynamic asteroid spawning and scoring
  • Start screen and game over restart system
